# Chapter 1
```sh
git init

git config --global user.email "email"
git config --global user.name "name"

git add file

git commit -m "message"

git log

git checkout "checksum"

git remote add "name" "addr"

git branch -M main

git push "remote_name" "branch_name"

git clone addr

git pull "remote_name" "branch_name"
```
